MikroGlass
==========

Looking glass system for Mikrotik routers.

[Online demo](http://ug.runtimeapp.com)

Features
--
* Ping
* Traceroute
* BGP route
* quick info (IP routing, BGP status)
* works over SSH, Linux and Windows supported (Putty Link)

Users
--
* [inAsset Networks](https://inasset.es/lg/)
